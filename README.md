# TODO List 应用

一个功能完整的待办事项管理 Web 应用，使用 React + TypeScript + Vite 构建。

## 功能特性

- ✅ 添加待办事项（包含标题，描述可选）
- ✅ 删除待办事项
- ✅ 标记待办事项完成/未完成
- ✅ 查看待办事项列表
- 💾 数据持久化（使用 localStorage）
- 🎨 现代化的 UI 设计

## 技术栈

- React 18
- TypeScript
- Vite
- CSS3

## 安装和运行

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
image.pngnpm run dev
```

3. 构建生产版本：
```bash
npm run build
```

4. 预览生产构建：
```bash
npm run preview
```

## 使用说明

1. **添加待办事项**：在输入框中输入标题，点击"添加描述"可以添加可选的描述信息，然后点击"添加待办"按钮。

2. **标记完成**：点击待办事项左侧的复选框来标记完成或未完成。

3. **删除待办**：点击待办事项右侧的删除按钮（🗑️）来删除该事项。

4. **查看列表**：所有待办事项会显示在列表中，顶部会显示统计信息（总计、已完成、待完成）。

## 数据存储

应用使用浏览器的 localStorage 来保存数据，刷新页面后数据不会丢失。
